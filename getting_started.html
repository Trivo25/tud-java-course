Making this Java thing work..

This guide will help you to get your development enviroment ready to go so we can start doing cool things right away!

In order to actually execute the code you wrote you need a compiler that basically translates your plain Java code into something your computer understands and can execute. The javac programm does exactly that - it compiles a plain .java file into something the so-called Java Virtual Machine understands.

To get everything working we need to install a Java development enviroment.
We recommend Java OpenJDK 11 by Adoptium Download (please select "Temurin 11 (LTS)" and the correct OS and architecture when downloading).
The installation depends on your operating system. Even tho we recommend Java 11 you are free to use any other version - but please keep in mind that things might be different!

For both Windows and Mac binaries and installers are available. Choose the one you are most familiar with - if you are new to this Java-thingie we recommend using the installer (.msi for Windows and .pkg for Mac) and just follow the instructions.
The installation guides for Windows, Mac and Linux are available right here.
